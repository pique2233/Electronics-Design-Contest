### 目标
构建智能视觉系统，实现高效图像处理与深度学习应用。

### 主要内容
- **图像采集**：摄像头选型及驱动开发
- **图像处理**：边缘检测、目标识别及跟踪算法优化
- **深度学习**：模型训练部署（TensorFlow/PyTorch）

## 项目目录

- `/hardware`：硬件设计文档
- `/firmware`：单片机固件代码
- `/vision`：视觉算法与模型代码
